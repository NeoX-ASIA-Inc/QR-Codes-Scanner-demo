# ユーザーストーリー:

- このアプリは✅のアイコンが２つあります。無効化ボタンが１つあります。
- アプリを開けて、QRコードスキャンを選びます。
- デフォルト色はグレーです。QRコードをスキャンすると、アイコン#1の色は赤くなり、アイコン#2の色は青くなります。
- ２つのQRコードは青いや赤くなったとき、ボタンが有効化されます。
- ボタンをクリックして、Youtubeビデオが開けます。
- アプリを閉じるとき、全てのアイコンは色がグレーにリセットします。
